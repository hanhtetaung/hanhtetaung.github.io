## Hospital Managment System 

The Hospital Management System is an ERP solution designed to manage the hospital operations using Laravel.

### Main Features

* Point of Sale: Process customer orders, manage transactions and generate sales reports
* Inventory Management: Manage medical supplies and track inventory levels
* Appointment: Schedule appointments with specific doctors and select available dates and times
* Patient Management: Register patients and track appointment histories.

### Used Technologies

- Bootstrap
- AdminLTE
- Laravel
- MariaDB
