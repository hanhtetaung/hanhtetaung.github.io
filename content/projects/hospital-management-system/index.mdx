## Hospital Managment System 

The Hospital Management System (HMS) is a software solution using Laravel designed for hospital operations, including point of sale, inventory management, patient management and appointment scheduling.

### Main Features

* Point of Sale
* Inventory Management
* Appointment Scheduling
* Patient Management

### Technologies

- Bootstrap
- AdminLTE
- Laravel
- MariaDB
